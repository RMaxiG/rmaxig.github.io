Documentación Técnica: Sitio Web "Turismo Jujuy"
1. Descripción General
Este proyecto consiste en una página web institucional para el Ministerio de Turismo de Jujuy. El objetivo es promocionar las cuatro regiones de la provincia (Quebrada, Yungas, Puna y Valles) mediante un diseño visualmente atractivo que incluye videos, imágenes y secciones interactivas.

2. Estructura del Proyecto
La carpeta del proyecto está organizada de la siguiente manera:

/root: Contiene el archivo principal index.html.

/css: Archivo styles.css con todos los estilos.

/img: Carpeta de recursos multimedia (fotos, logos y el video Jujuy.mp4).

/paginas: (Proyectado) Para archivos como Destinos.html, Contacto.html, etc.

3. Tecnologías Utilizadas
HTML5: Estructura semántica del contenido.

CSS3: Diseño responsivo y animaciones (usando Flexbox y Grid).

Multimedia: Uso de etiquetas <video> para el Hero y etiquetas <img> para las galerías.

4. Secciones de la Página
A. Encabezado (Header)
Contiene el logo de la institución y un menú de navegación.

Detalle técnico: Se utilizó la etiqueta <details> y <summary> para crear un menú desplegable nativo sin necesidad de JavaScript complejo.

B. Pie de Página (Footer)
Dividido en tres partes principales:

Newsletter: Un formulario simple para captar emails.

Redes Sociales: Enlaces externos a las cuentas oficiales de turismo.

Mapa: Un iframe que vincula a Google Maps para mostrar la ubicación física de las oficinas.

5. Decisiones de Diseño
Interactividad: Se buscó que el usuario tenga una experiencia visual rápida.

Navegación: El menú está pensado para ser accesible tanto en PC como en dispositivos móviles.

Colores: Se respetan los colores institucionales y se usan overlays oscuros sobre el video para que el texto sea legible.
